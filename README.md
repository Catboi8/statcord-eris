# statcord.js

# Init
## This should be placed inside the `ready` event only!!
```js
let statcord = require("statcord.js");

let statClient = new statcord("KEY", DiscordClient);
```
### Posting Once (Without Auto Post)
```js
statClient.post()
```

### Auto Posting
```js
statClient.autoPost()
```
